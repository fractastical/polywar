Polygon Warfare Tutorial

Basics

Build a network of polygons to generate resources and defend against enemies.

Controls

- Press T to toggle between Producer/Combat modes

- Use 3-9 to select polygon sides

- Click to place polygons

- Right-click to move polygons

Polygon Types

Producers: Generate resources over time based on sides

Combat: Automatically spawn fighters to defend territory

Strategy

- Start with producer polygons to generate resources

- Build combat polygons near edges to defend

- More sides = more cost but more power

- Keep producers protected behind combat units

Combat Rules

- Equal sides destroy each other

- One more side beats lower number

- Other combinations bounce off
